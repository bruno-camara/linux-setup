# Linux Setup

>Currently using Ubuntu 22.04

Scripts to install and configure my linux programs.

> Obs: The Python version that comes with Ubuntu 22.04 is 3.10.12

## How to run
Give permissions in order to run the scripts
```bash
chmod -R +x ./install
chmod -R +x ./config
```

## Programs included
- Chrome
- Visual Studio Code
- Vim
- Git
- Node and npm
- Docker
- R and RStudio
- docker compose

## Python Libraries
- virtualenv
- pandas
